A tiny macOS Quick Look extension. Select a .md file in Finder, press spacebar, and read it the way it was meant to look — headings, code, tables, and task lists included.

What it does

Everything you'd expect, natively.

Rich rendering

Headings, lists, task lists, blockquotes, links, bold, italic, strikethrough and inline code.

Syntax-highlighted code

Fenced code blocks are highlighted with a clean GitHub-style theme, light and dark.

Real tables

GitHub-flavored Markdown tables render as native AppKit tables — not raw pipes.

Sandboxed & offline

Runs in a sandboxed Quick Look extension with no network access. A menu-bar helper, no Dock clutter.

Get started

Install in three steps.

Download & open the disk image

Grab MarkdownQuickLook.dmg and drag the app into your Applications folder.

Launch it once

Open Markdown QuickLook from Applications. It lives in your menu bar and registers the Quick Look extension.

Press spacebar

Select any .md file in Finder and hit space. If a previous preview is cached, use the menu-bar item → Reset Quick Look Cache.

Not notarized yet: on first launch macOS may say the app is from an unidentified developer. Right-click the app → Open → Open, or run xattr -dr com.apple.quarantine /Applications/MarkdownQuickLook.app. Prefer building from source? See the README.
