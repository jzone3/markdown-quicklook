# Tables and Code

## A table

| Element | Supported | Notes                     |
| :------ | :-------: | :------------------------ |
| Tables  |    yes    | With column alignment     |
| Code    |    yes    | Syntax highlighted        |
| Images  |    yes    | Local and remote          |

## A code block

```swift
func greet(_ name: String) -> String {
    return "Hello, \(name)!"
}

print(greet("Markdown"))
```

## Inline formatting

You can mix **bold**, *italic*, `inline code`, and ~~strikethrough~~ in a
single line.
