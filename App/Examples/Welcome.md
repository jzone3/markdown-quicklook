# Welcome to Markdown QuickLook

Select this file in Finder and press the **spacebar** to see it rendered.

Markdown QuickLook renders Markdown with the same engine used everywhere on
GitHub, so your notes, READMEs, and docs look the way you expect.

## What you get

- GitHub-flavored Markdown (tables, task lists, strikethrough)
- Syntax-highlighted code blocks
- Automatic light and dark appearance

> Tip: try the other files in this folder, then press space on your own
> `.md` files anywhere in Finder.
